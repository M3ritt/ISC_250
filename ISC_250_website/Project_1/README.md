This is project 1 and is a replication of Tetris made only using JavaScript and can be played using the Tetris page on the website. 

The controls of the game are: 
   - 'A' - left 
   - 'D' - right 
   - 'W' - hard drop
   - 'Enter' - rotate
   - 'Space' - hold
